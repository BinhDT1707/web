Introduction to HTML

HTML (Hypertext Makup Language) is used to create document on the World Wide Web. It is simply a collection of certain key words call "Tag" that are helpful in writing the document to be displayed using a browser on Internet. HTML was developed by Tim Berners-Lee in 1992.
